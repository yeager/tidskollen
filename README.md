# Tidskollen

Visual time timer for classrooms and children with autism.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install tidskollen
```

### Fedora/RPM
```bash
sudo dnf install tidskollen
```
